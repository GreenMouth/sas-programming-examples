# SAS Programming Examples

SAS programming examples working through exercises from *The Little SAS Book*.

## Overview

This repository contains SAS code written while working through *The Little SAS Book*, a widely used introduction to SAS programming. The examples are organized by chapter and cover core data-step and procedure techniques.

## Contents

| Item | Description |
|------|-------------|
| `Chap 1&2 Code.sas` | SAS code for Chapters 1 and 2. |
| `Chap 3 Code.sas` | SAS code for Chapter 3. |

## Datasets

The datasets used by these examples are provided by SAS and are available from the official support site:
http://support.sas.com/kb/33/486.html

## Requirements

- A SAS environment (e.g., SAS 9.x, SAS OnDemand for Academics, or SAS Studio)

## Usage

Open the `.sas` files in your SAS environment and submit the code. Ensure the accompanying datasets are available in your working library first.
